## 🚀 Sobre o Projeto

Este projeto foi desenvolvido utilizando **Python + Tkinter** com o objetivo de transformar um simples pedido em uma experiência divertida, criativa e memorável.

Aqui, a pessoa não apenas responde… ela **joga uma batalha contra o botão "Não"** 😈

---

## 🎮 Funcionalidades

✨ Botão **"Não"** que foge do mouse  
❤️ Botão **"Sim"** que ativa uma resposta especial  
💬 Mensagens dinâmicas e engraçadas  
💘 Animação com corações na tela  
🎨 Interface simples, mas charmosa  

---

## 🧠 Conceitos aplicados

- Manipulação de eventos (`<Motion>`)
- Interface gráfica com Tkinter
- Lógica de interação com usuário
- Uso de aleatoriedade para comportamento dinâmico
